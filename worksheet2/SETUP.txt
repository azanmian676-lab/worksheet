Azan Tariq - SimpleOS Setup

Tools needed:
- NASM
- i686-elf-gcc (cross compiler)
- QEMU

Setup steps:
1. Install NASM
2. Install i686-elf cross compiler
3. Install QEMU
4. Add all to PATH

To run:
run-direct.bat

This starts SimpleOS in QEMU. You'll see a command prompt where you can type:
- say <text>
- wipe
- commands
- ver
- stop

Close QEMU window when done.
